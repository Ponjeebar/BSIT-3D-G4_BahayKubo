MENU DISH IMAGES FOLDER
======================

This folder contains images for menu dishes. Each image should be named after its dish.

Image Naming Convention:
- Use lowercase letters
- Replace spaces with hyphens (-)
- Remove special characters
- File extension: .jpg, .png, or .webp

Example:
- "Lumpia Shanghai" → "lumpia-shanghai.jpg"
- "Chicken Adobo" → "chicken-adobo.jpg"
- "Halo-Halo" → "halo-halo.jpg"
